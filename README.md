# Double_YAOTYO_Reversi

5/20, 21の技育CAMPで作成したwebアプリケーションです。webアプリの概要はOpenAIと対戦できるオセロです。
しかし特筆すべき点は、このオセロが『ダブル八百長対決』である点です。つまり、UserとAIがお互い負けようとしなければなりません！
